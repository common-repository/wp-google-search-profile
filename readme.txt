== Description ==

 When Clients / Supports Search for Your organization on Google, then to see the Best Possible Results on page. so add your social media profiles to your website’s Google Search Profile.




== Installation ==

1. Upload "Wp plugin Google Search Profiles " folder to your " /wp-content/plugins/ directory " 
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Add Your profiles information Go to Admin menu (Admin > admin-menu > GSP Options) 




== Frequently Asked Questions == 

== How it Work ?

select type of your person or Organization in select type field.
Add the name of your organization or person to the ”name” field.
Add your social media links to the social media field , then save. 

== Where can I find the settings ?

it just go to admin menu then show "GSP Options"
